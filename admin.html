Admin Dashboard - Edit Website Content

Welcome to the CMS

This is the content management system for the Abiding Place Fellowship website. Log in below to edit website content without any coding knowledge.

🔐 Login to Edit Content
What you can edit:
  • 👥 Leadership Team (names, roles, bios, photos)
  • 📅 Events & Services (dates, locations, descriptions)
  • 🏛️ Ministries (kids, youth, adults programs)
  • 📞 Contact Information (phone, email, address)

How It Works

1. Click Login

Use your Google account to log in securely.

2. Edit Content

Fill out simple forms to update team, events, or contact info.

3. Publish

Click "Publish" and changes go live in ~30 seconds.

Quick Tips

  • All changes are saved automatically as you type
  • You can preview changes before publishing
  • Use the "Status" panel to see what's published
  • Contact abidingplaceministries@sympatico.ca for help
