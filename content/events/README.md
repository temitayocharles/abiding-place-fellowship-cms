# Events publishing rules

The repository intentionally contains no unverified future special events.

Publish only when the church confirms:

- exact event name
- date and start/end time
- location and entrance
- organiser or ministry owner
- registration requirements
- child-safety or accessibility details
- cancellation contact
- whether photography is expected

The Tuesday and Wednesday meeting times are consistently published. Their study topics conflict across legacy pages, so the current topic must be confirmed before publication.
